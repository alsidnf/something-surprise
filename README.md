# Something Surprise

GitHub Pages에서도 이 정도까지 가능하다는 느낌을 보여주는 인터랙티브 웹 프레젠테이션입니다.

## What is inside

- PPT처럼 한 장씩 넘어가는 스크롤 스냅 데크
- 마우스로 돌려보는 Three.js 3D 결정체
- 포인터 위치에 반응하는 Canvas 입자장
- 클릭 파동에 반응하는 수중 연못 생태계
- 슬라이드 안에서 바로 플레이하는 미니 게임
- 실시간으로 움직이는 제너레이티브 피날레

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The Vite base path is set to `/something-surprise/` for GitHub Pages deployment from this repository name.
